# HaloFibersGit

## Deployment

This repository is configured for automatic deployment to GitHub Pages.

### How it works

- The site is automatically deployed when changes are pushed to the `main` branch
- You can also manually trigger a deployment from the Actions tab

### Setup

To enable GitHub Pages deployment for a fork of this repository:

1. Go to your repository's **Settings** > **Pages**
2. Under "Build and deployment", select **GitHub Actions** as the source
3. The deployment workflow will run automatically on the next push to `main`

### Manual Deployment

1. Go to the **Actions** tab in your repository
2. Select the "Deploy to GitHub Pages" workflow
3. Click "Run workflow" to trigger a manual deployment
